# Notice and attribution

This repository is an independent Python educational implementation inspired by the public project:

- `vishalmysore/agenticknowledgegraph`

The new implementation preserves the broad teaching themes—five example domains, progressive graph queries, and KuzuDB—while introducing a different package architecture, new synthetic datasets, validation, tests, a command-line interface, GraphRAG context construction, and a reusable agent loop.

No warranty is provided. Fraud, health, and risk examples are synthetic and are not production decision systems.
